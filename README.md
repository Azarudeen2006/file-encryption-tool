# 🔐 File Encryption Tool

A Python-based **File Encryption and Decryption Tool** that protects files by converting readable data into encrypted format using the **Fernet symmetric encryption algorithm**. The project also includes a simple graphical user interface (GUI) using **Tkinter**.

---

## 📌 Project Overview

The File Encryption Tool is a cybersecurity project designed to secure sensitive files from unauthorized access.

Users can:
- Select files using a graphical interface
- Encrypt files securely
- Decrypt encrypted files
- Restore the original file using the secret key
