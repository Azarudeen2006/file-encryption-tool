# 🔐 File Encryption & Decryption Tool

## 📌 Project Overview

File Encryption & Decryption Tool is a cybersecurity application developed using Python. 
This tool helps users protect sensitive files by converting them into an encrypted format and allows authorized users to restore them using a secure key.

The project uses cryptographic techniques to provide data confidentiality and prevent unauthorized access.

---

## 🚀 Features

✅ Encrypt files securely  
✅ Decrypt encrypted files  
✅ Protect sensitive documents and data  
✅ Secret key based security  
✅ Simple graphical user interface  
✅ Prevent unauthorized file access  

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- Cryptography Library
- Fernet Encryption Algorithm
